# PydanticAI 주요 특징 요약

PydanticAI는 생성형 AI를 활용한 프로덕션급 애플리케이션 및 워크플로우를 신속하고 안정적으로 구축할 수 있도록 설계된 파이썬 에이전트 프레임워크입니다. FastAPI의 직관적이고 인체공학적인 설계를 GenAI 개발에 도입하는 것을 목표로 합니다.

## 주요 특징

1. **Pydantic 팀의 신뢰성**: 업계 표준인 Pydantic 검증 로직을 직접 활용합니다.
2. **모델 독립성 (Model-agnostic)**: OpenAI, Anthropic, Gemini 등 주요 제공업체 및 다양한 모델을 폭넓게 지원하며, 커스텀 모델 구현도 용이합니다.
3. **원활한 관측성 (Observability)**: Pydantic Logfire와 긴밀하게 통합되어 실시간 디버깅, 성능 모니터링, 추적 및 비용 관리가 가능합니다.
4. **완벽한 타입 안전성 (Type-safe)**: 타입 힌트를 적극 활용하여 IDE 자동 완성 및 오류 방지를 극대화합니다.
5. **강력한 평가 기능 (Evals)**: 에이전트 시스템의 성능과 정확도를 체계적으로 테스트하고 모니터링할 수 있습니다.
6. **확장성 (Extensible)**: 도구, 지침, 설정을 재사용 가능한 단위로 구성할 수 있으며, YAML/JSON 정의를 지원합니다.
7. **MCP 및 에이전트 간 통신**: Model Context Protocol(MCP) 및 Agent2Agent 기술을 지원하여 외부 도구 연동 및 에이전트 간 상호 운용성을 제공합니다.
8. **인간 개입(Human-in-the-Loop) 도구 승인**: 도구 실행 시 사용자 승인 프로세스를 쉽게 구현할 수 있습니다.
9. **지속 가능한 실행 (Durable Execution)**: API 오류나 프로세스 재시작에도 진행 상태를 보존하며 안정적인 워크플로우를 지원합니다.
10. **스트리밍 출력**: 구조화된 출력을 실시간으로 스트리밍하며 즉각적인 검증을 수행합니다.
11. **그래프 지원**: 복잡한 제어 흐름을 타입 힌트를 사용하여 그래프 구조로 정의할 수 있습니다.
